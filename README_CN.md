FFmpeg 与 ImageMagick 的终极前端。**Universal Converter (UCT)** 在一个简洁、智能的拖拽式界面中，让您以专业级的水准轻松转换视频、音频和图像。

---

## 核心功能

### 基于行业标准构建
利用 FFmpeg（用于视频/音频）和 ImageMagick（用于图像）久经考验的强大性能与稳定性，无缝处理您的所有媒体文件。

### 智能批量处理
想一次性转换 50 个文件？没问题。UCT 能够轻松应对大批量文件队列，并内置了**智能筛选逻辑**：如果您同时拖入 8 个视频和 2 张图片，UCT 会自动导入数量较多的视频，并提示您分开处理图片，彻底告别队列冲突。

### 拖拽即用
抛弃繁琐的文件选择器吧。只需点击导入区域或直接从资源管理器将文件拖入应用，即可瞬间建立您的转换队列。

### 专业级参数控制
别再忍受只有“低/中/高”的简陋画质选项了。UCT 为您解锁媒体引擎的全部潜能：
* **常规选项:** 调节播放速度、音量、旋转视频、调整分辨率以及图像降噪。
* **编解码器设置:** 选择如 H.264, ALAC 或 ProRes 等行业标准编码器，并微调预设 (Preset)、调优 (Tune)、配置 (Profile) 及像素格式。
* **命令实时预览:** 在点击开始之前，即可实时预览输入和输出文件的完整命令行。

### 完全透明化 (内置终端)
拒绝黑箱操作。UCT 将 FFmpeg / ImageMagick 的原始输出流实时投射到内置的终端窗口中。这非常适合用于调试复杂的转换任务，或者验证您的自定义参数是否被正确应用。
