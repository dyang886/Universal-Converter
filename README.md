The ultimate frontend for FFmpeg and ImageMagick. **Universal Converter (UCT)** allows you to convert video, audio, and images at a professional level—all within a clean, smart, and intuitive drag-and-drop interface.

---

## Core Features

### Built on Industry Standards
Seamlessly process all your media files using the proven power and reliability of FFmpeg (for video/audio) and ImageMagick (for images).

### Smart Batch Processing
Want to convert 50 files at once? No problem. UCT handles massive queues effortlessly with intelligent filtering logic. If you drag in 8 videos and 2 images simultaneously, UCT automatically imports the majority file type (videos) and prompts you to process the images separately, completely eliminating queue conflicts. 

### Drag-and-Drop Interface
Forget clunky file pickers. Simply click the import zone or drag files directly from Windows Explorer into the app to instantly build your conversion queue.

### Professional-Grade Control
Stop settling for basic "Low/Medium/High" quality toggles. UCT unlocks the full potential of your media engines:
* **General Settings:** Adjust playback speed, volume, rotation, resolutions, and apply image denoising.
* **Codec Configuration:** Select industry-standard encoders like H.264, ALAC, or ProRes, and fine-tune your presets, tunes, profiles, and pixel formats.
* **Live Command Preview:** See the exact generated command lines for your input and output files in real-time before you hit start.

### Complete Terminal Transparency
No black-box operations here. UCT projects the raw output streams from FFmpeg and ImageMagick directly into a built-in terminal window. This is perfect for debugging complex conversion tasks or verifying that your custom parameters are being applied correctly.
