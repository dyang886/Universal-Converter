FFmpeg 與 ImageMagick 的終極前端。**Universal Converter (UCT)** 在一個乾淨、智慧的拖曳式介面中，讓您以專業級的水準輕鬆轉換影片、音訊和圖片。

---

## 核心功能

### 基於業界標準建構
利用 FFmpeg（用於影片/音訊）和 ImageMagick（用於圖片）久經考驗的強大效能與穩定性，無縫處理您的所有媒體檔案。

### 智慧批次處理
想一次性轉換 50 個檔案？沒問題。UCT 能夠輕鬆應付大量檔案佇列，並內建了**智慧篩選邏輯**：如果您同時拖入 8 個影片和 2 張圖片，UCT 會自動匯入數量較多的影片，並提示您分開處理圖片，徹底告別佇列衝突。

### 拖曳即用
拋棄繁瑣的檔案選擇器吧。只需點擊匯入區塊或直接從檔案總管將檔案拖入應用程式，即可瞬間建立您的轉換佇列。

### 專業級參數控制
別再忍受只有「低/中/高」的簡陋畫質選項了。UCT 為您解鎖媒體引擎的全部潛能：
* **一般設定:** 調整播放速度、音量、旋轉影片、調整解析度以及圖片降噪。
* **編解碼器配置:** 選擇如 H.264, ALAC 或 ProRes 等業界標準編碼器，並微調預設 (Preset)、調優 (Tune)、配置 (Profile) 及像素格式。
* **命令即時預覽:** 在點擊開始之前，即可即時預覽輸入和輸出檔案的完整命令列。

### 完全透明化 (內建終端機)
拒絕黑箱作業。UCT 將 FFmpeg / ImageMagick 的原始輸出串流即時投射到內建的終端機視窗中。這非常適合用於除錯複雜的轉換任務，或者驗證您的自訂參數是否被正確應用。

1. 前往我們的 [最新發布 (Latest Release)](https://github.com/dyang886/Universal-Converter/releases) 頁面。
2. 下載最新的 Windows (64位元) 安裝檔。
3. 執行安裝程式並按照螢幕上的指示進行操作。
4. 啟動 UCT，開始轉換！
